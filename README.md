## How To Use
```bash
# Clone this repository
$ git clone https://github.com/danirafsanjani/ngebus-cak

# Go into the repository
$ cd ngebus-cak

# Install dependencies
$ npm install  --global expo-cli

# Run the app
$ expo start
```
